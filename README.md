# EscapePlan   EscapePlan é uma API RESTful desenvolvida em Node.js com Express, focada em auxiliar usuários a alcançarem objetivos financeiros reais por meio do planejamento de metas.  O sistema aplica o conceito de **abstração**, permitindo que grandes objetivos financeiros sejam divididos em metas menores, facilitando o acompanhamento e a execução do plano financeiro.  ##  Funcionalidades - Cadastro de usuários - Criação, edição, exclusão e visualização de objetivos financeiros - Planejamento e acompanhamento de metas vinculadas a cada objetivo - Controle de renda fixa mensal - API estruturada seguindo o padrão MVC  ##  Tecnologias Utilizadas - Node.js - Express - MySQL - dotenv - JavaScript (ES Modules)  ##  Arquitetura - Padrão MVC (Model, Controller, Repository) - Pool de conexões com MySQL - Separação de responsabilidades entre camadas - API RESTful  ##  Objetivo do Projeto Este projeto foi desenvolvido com fins educacionais e de portfólio, com foco no aprendizado de: - Arquitetura backend - Boas práticas de organização de código - Integração com banco de dados relacional - Construção de APIs escaláveis  ##  Status do Projeto:  Em desenvolvimento  ---
