API REST em Node.js e Express para planejamento de objetivos financeiros, estruturada em MVC e integrada ao MySQL.

Funcionalidades:
    Cadastro de usuários
    Criação, edição, exclusão e visualização de objetivos financeiros
    Planejamento e acompanhamento de metas vinculadas a cada objetivo
    Controle de renda fixa mensal
    API estruturada seguindo o padrão MVC

Arquitetura:
    Padrão MVC (Model, Controller, Repository)
    Pool de conexões com MySQL
    Separação de responsabilidades entre camadas
    API RESTful
    
Tecnologias Utilizadas:
    Node.js
    Express
    MySQL
    dotenv
    JavaScript (ES Modules)
